## Steps to get the code and run the tests locally
1. `git clone https://github.com/hi2suresh/planit-aut.git`
2. `cd planit-aut`
3. `npm install`
4. `npm run test:all`

## Generate Allure report
Navigate to project root folder and run `allure serve`
